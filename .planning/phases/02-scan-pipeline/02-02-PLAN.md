**Step 2 — Create src/extractor.py:**

Two public functions:

1. `extract_text(file_path: str) -> str | None` — Dispatches by file extension:
   - `.txt`, `.log`: `_extract_text_file(path)` — encoding fallback chain: try `utf-8-sig`, on UnicodeDecodeError try `cp1252`, final fallback `latin-1` (never raises). Return file content as string.
   - `.csv`: `_extract_csv(path)` — same encoding fallback chain. Read with `csv.reader`, join all cell values with spaces per row, newlines between rows. Return as string.
   - `.docx`: `_extract_docx(path)` — `docx.Document(path)`, join `[p.text for p in doc.paragraphs]` with newlines. Also iterate `doc.tables` and extract cell text. Catch `BadZipFile`, `KeyError`, `Exception`.
   - `.xlsx`, `.xls`: `_extract_xlsx(path)` — `openpyxl.load_workbook(path, read_only=True, data_only=True)`. Iterate active sheet rows (max 1000 rows), join cell values with spaces. Catch `InvalidFileException`, `BadZipFile`, `Exception`.
   - `.pdf`: `_extract_pdf(path)` — First check `os.path.getsize(path) > 20 * 1024 * 1024` and return None if over 20 MB (per SCAN-03 guard). Use `pdfplumber.open(path)` as context manager, iterate pages up to 50, join `page.extract_text()` results. Catch all exceptions.
   - Unknown extension: return None.
   
   Every extractor branch is wrapped in `try/except Exception` — log the error with `logging.warning(f"Failed to extract {file_path}: {e}")` and return None. Per SCAN-05: one bad file never crashes the scan.

2. `extract_headers(file_path: str) -> list[str]` — For tabular files only:
   - `.csv`: Read first row with same encoding fallback, return as list of strings.
   - `.xlsx`, `.xls`: `openpyxl.load_workbook(path, read_only=True, data_only=True)`, read first row of active sheet, return cell values as list of strings.
   - Other extensions: return empty list.
   
   Wrapped in `try/except Exception` returning empty list on failure.

Module-level constant: `SUPPORTED_EXTENSIONS = {".docx", ".xlsx", ".xls", ".csv", ".pdf", ".txt", ".log"}`

Import guard: check extension against SUPPORTED_EXTENSIONS at top of extract_text(); return None for unsupported.
