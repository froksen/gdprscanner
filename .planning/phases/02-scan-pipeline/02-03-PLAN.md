**Create src/scanner.py:**

```python
class ScanEngine:
    def __init__(self, event_queue: queue.Queue, config_store: ConfigStore, ignore_set: set[str] | None = None) -> None:
```

`ignore_set` parameter: a `set` of absolute file paths to skip (O(1) lookup per IGNR-02). Default to empty set. In Phase 3, this will be loaded from ignorelist.json. For now, accept as constructor parameter.

**Method `run_scan(self) -> None`:**

1. Read config snapshot: `config = self.config_store.get_config()`
2. Extract: `folders = config["scan_folders"]`, `age_days = config["file_age_days"]`, `file_types = set(config["file_types"])`
3. Compute age cutoff: `cutoff_ts = time.time() - (age_days * 86400)`
4. Counters: `files_scanned = 0`, `findings_count = 0`
5. For each folder in folders:
   - `os.walk(folder)` (default followlinks=False per Pitfall 14)
   - For each file:
     a. **Extension filter (FILT-02):** `pathlib.Path(f).suffix.lower()` in `file_types`? Skip if not.
     b. **Full path:** `file_path = os.path.join(dirpath, filename)`
     c. **Ignore list filter (FILT-04):** `os.path.abspath(file_path) in self.ignore_set`? Skip if yes.
     d. **Age filter (FILT-01):** `os.path.getmtime(file_path) < cutoff_ts`? Skip if file is NEWER than cutoff (we want OLD files). Logic: if `mtime > cutoff_ts`, file is too young, skip.
     e. `files_scanned += 1`
     f. **Per-file try/except** wrapping steps g-k:
     g. `file_age_days = int((time.time() - os.path.getmtime(file_path)) / 86400)`
     h. **Filename detection (FILT-03):** `keywords = detect_pii_filename(file_path)`. For each keyword, emit `FindingEvent(file_path=file_path, file_age_days=file_age_days, detection_rule="filename_keyword", matched_value=keyword)` and increment findings_count.
     i. **Content extraction:** `text = extract_text(file_path)`. If None, continue to next file.
     j. **Header extraction (for .csv, .xlsx, .xls):** `headers = extract_headers(file_path)`. If headers, `pii_headers = detect_pii_headers(headers)`. For each, emit FindingEvent with detection_rule="pii_header".
     k. **Content detection:** Run `detect_cpr(text)`, `detect_email(text)`, `detect_phone(text)`. For each match list, emit FindingEvent with appropriate detection_rule ("cpr", "email", "phone") and matched_value. Increment findings_count.
     l. **Exception handler:** `except Exception as e: logging.warning(f"Skipping {file_path}: {e}")` — continue loop.
   - Insert `time.sleep(0.01)` between files to avoid I/O starvation (Pitfall 8).
6. Emit `ScanCompleteEvent(files_scanned=files_scanned, findings_count=findings_count)`.
