```python
def _open_config_dialog(self):
    """Open config dialog as Toplevel child of hidden root."""
    # If dialog already exists and is visible, bring it to front
    if self._config_dialog is not None:
        try:
            self._config_dialog.dialog.lift()
            self._config_dialog.dialog.focus_force()
            return
        except tk.TclError:
            # Dialog was destroyed, create new one
            self._config_dialog = None

    self._config_dialog = ConfigDialog(self.root, self.config_store)
```

3. Ensure `self._config_dialog = None` is in `__init__` (should already exist from Plan 02).

This ensures:
- Per UI-SPEC interaction contract: dialog opens via `tk.Toplevel(root)`
- Per UI-SPEC: only one dialog instance at a time (if already open, lifts existing)
- Per D-06: dialog is Toplevel, not new Tk
- All dialog creation happens on tkinter thread (this method is called from `_handle_event` which runs in the tkinter `after()` callback)
